The Caesar Cipher

About The Project

The Caesar Cipher, codenamed, 'Project 2C', is classic encryption and decryption utility powered by both Python and interactive web technologies.

My motivation

The Caesar Cipher is one of the oldest and most iconic encryption methods. I built this project to help users experiment with how shifting letters can transform messages, making cryptography easy to understand and fun to use.

Features

  • Encrypts and decrypts text using the Caesar Cipher algorithm
  • Handles both uppercase and lowercase letters
  • Input validation for shift values and mode
  • Interactive demo for instant results

Tech Stack

  • Python, the raw code
  • JavaScript for browser interactivity
  • HTML and CSS for clean, sleek aesthetic UI
