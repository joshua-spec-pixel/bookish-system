Productivity-Time Dilation Calculator

About The Project

The Productivity-Time Dilation Calculator, codenamed 'Project P', is playful exploration of how focus transforms our perception of time at work.

My motivation

I was hell-bent on implementing the concept of time-dilation in daily life. I had noticed that a highly focused hour of work seems to stretch longer than a distracted one?
Project P quantifies this phenomenon, inspired by the concept of time dilation from physics; except here, the “speed” is your focus.

Features

  • Calculates perceived (dilated) work time based on your focus
  • Validates user input for realistic focus values
  • Playful, science-inspired interface

Tech Stack

  • Python
  • NumPy library for mathematical operation
  • JavaScript to enable web interactivity
  • HTML and CSS for stylizing the UI.
