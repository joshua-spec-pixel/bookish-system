Project V: The Vintage Journal-Timer

About The Project

A blend of classically designed journaling and timing systems to make deep work feel ritualistic
Inspired by chess discipline and temporal focus

What Problem This Solves

Most productivity tools focus on urgency and speed.
Project V restructure work, shifting it from hectic, frantic, grunt work to disciplined, reflective and purposeful tasks, helping users escape dopamine loops.

Key Screens

The Menu Window

Entry point for a less grunt and more cermonious start

The Journal Window

Endless journalling page to reflect on thoughts and ideas
Save them and revisit anytime

The Timer Window

Modus operandi of pomodoro timer for disciplined, deep work and not for urgency

Why it's Designed This Way

Influenced by the idea of persistent habits like journalling and time keeping,
I built this tool to encourage consistency over intensity. The vintage aesthetic reinforces calm focus and purposeful action.

Features

  • Interactive timer for work and break
  • Journal, annotate, save and load notes
  • Old money-retro styled interface, easy-to-use

Tech Stack

  • Python (core logic)
  • Tkinter library (GUI system)
  • Threading (concurrent window handling)
  • Time library (precise timing)
