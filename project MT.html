About The Project

The Morse Code Translator codenamed 'Project MT', is a digital toolkit for anyone interested in Morse code communicationâ€”whether you're a hobbyist, student, or explorer of classic cryptography.

My motivation

As someone captivated by the elegance of Morse code and its historical significance, I wanted to create a tool that not only encodes and decodes messages but also makes learning and experimenting with Morse code engaging and accessible.

Features

  • Encode text to Morse code and decode Morse code back to text
  • Real-time interactive interface for practicing Morse code
  • Visual and audio output options for signals
  • Save and load messages for future reference
  • User-friendly, modern design with intuitive controls

Tech Stack

  • Python as raw code
  • HTML and CSS to design the GUI
  • JavaScript to run a web-friendly UI
