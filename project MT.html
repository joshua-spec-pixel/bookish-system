Morse Code Translator

About The Project

Project MT is a digital toolkit for anyone interested in Morse code communication, whether you're a hobbyist, a student, or an explorer of classic cryptography.

What Problem This Solves

Morse code is an essential survival skill and good hobby to learn and with the help of Project MT, the interactive and easily operable UI, users can practice ciphering and deciphering this algorithm.

Key Screen

Why it's Designed this Way

This interactive Morse code translator combines real-time encoding/decoding with educational features. Project MT makes this historic communication system meaningful to learn, demonstrating both my interest in digital preservation and UX design.

Features

  • Encode and decode Morse code
  • Real-time interactive interface for practicing Morse code
  • User-friendly, modern design with intuitive controls

Tech Stack

  • Python (core logic, click 'Enter the archive' to access code)
  • JavaScript to run a web-friendly UI
  • HTML and CSS to design the GUI
