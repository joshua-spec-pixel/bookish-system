About The Project

The Vintage Journal-Timer codenamed 'Project V', is a timekeeping companion for chess enthusiasts, crafted with precision and a hint of flair.

My motivation

As someone fascinated by both chess and the art of timing, I wanted a tool that could elegantly track moves, journal thoughts, and add a dash of personality to every work and game.

Features

  • Interactive timer for work and break
  • Journal, annotate, save and load notes
  • Old money-retro styled interface, easy-to-use

Tech Stack

  • Python
  • Tkinter library to design the GUI
  • Threading library to run multiple windows indepenently and simultaneously
  • Time library for the timer window
